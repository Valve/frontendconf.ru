Browser Fingerprinting?

Идентификация пользователей без помощи cookie

История покупок или просмотренных страниц
Внутренняя аналитика
Показ персонализированной рекламы
Недостатки идентификации
 через cookie
Пользователи часто очищают cookie
Многие работают в инкогнито режиме
Различные плагины препятствуют сохранению
информации в cookie
Evercookie - попытка решения 
проблем с cookie подходом
HTTP Cookies

Isolated Storage (Silverlight)
PNG cookie encoding/decoding with canvas
HTML5 LocalStorage, SessionStorage
ETag storage
Java storage
Уязвимости в Java апплетах (CVE-2013-0422 exploit)
Evercookie - все хорошо, но...

FingerprintJS (первая версия)
  • Создан в 2012 году
  • Не использует cookie или другие методы хранения
  • Работает в private режиме (!)
  • Размер 800 байт (gzipped)
  • Нет зависимостей
  • Используется в Baidu, Mastercard, Whitehouse.gov и других
  • .. обладает рядом недостатков (о них позже)
Принципы работы FingerprintJS
  • UserAgent
  • Browser language
  • DateTime offset
  • Screen resolution
  • Color depth
  • HTML5 technologies support
  • Plugins info
  • Canvas fingerptint
Пример получаемых свойств
  • Mozilla/5.0 (Macintosh; Intel Mac......42.0.2311.90 Safari
  • ru-RU
  • -180 (datetime offset)
  • 24 (color depth)
  • HTML5 technologies support (true,true,false,true ...)
  • ....
